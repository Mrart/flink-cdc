    This documentation is for an unreleased version of Apache Flink CDC. We recommend you use the latest stable version.

    Pipeline Connectors #

    Flink CDC 提供了可用于 YAML 作业的 Pipeline Source 和 Sink 连接器来与外部系统交互。您可以直接使用这些连接器，只需将 JAR 文件添加到您的 Flink CDC 环境中，并在您的 YAML Pipeline 定义中指定所需的连接器。

    Supported Connectors #

    Develop Your Own Connector #

    如果现有的连接器无法满足您的需求，您可以自行开发自己的连接器，以将您的外部系统集成到 Flink CDC 数据管道中。查阅 Flink CDC APIs 了解如何开发您自己的连接器。
